# Superpowers Docs

This subtree is a historical design/spec archive. Most files were written for a
specific decision point or implementation pass and are not automatically updated
as the code changes.

Use these files for rationale and original intent. For current project status,
use the repo-root `TODO.md`, `docs/ROADMAP-STATUS.md`, and any current
operations runbook in `docs/operations/`.

Before treating a claim in `specs/` or `plans/` as current, re-check it against
the present source tree and promote the verified work item into the active TODO
or roadmap.

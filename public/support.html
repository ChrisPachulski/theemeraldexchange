Support

The Emerald Exchange is a client for your own self-hosted media server. Most connection and library issues can be narrowed down by checking the device, server address, and server together.

Apple deviceConfirm Wi-Fi or Ethernet is connected.
Server addressUse the complete HTTPS or local-network URL.
Self-hosted serverConfirm it is online and reachable from that network.

First checks

Before you contact support

  • Open the server address in a browser on the same network and confirm the server responds.
  • Check the full address, including https:// and any custom port or path.
  • If pairing expired, request a new invite code from your server administrator.
  • For Bonjour discovery, keep the Apple device and server on the same local network and allow Local Network access in system Settings.
  • For playback trouble, note whether the title is video on demand, live TV, or a download, and whether another device can play it from the same server.

Useful detail

What to include

Include the app version shown in Account, the device model and operating-system version, the type of server you use, and the exact step that failed.

Do not send passwords, authentication tokens, invite codes, private media links, provisioning profiles, or signing keys. A redacted screenshot is welcome when it makes the problem clearer.
